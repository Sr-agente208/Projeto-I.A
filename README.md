## 🚀 Compilação Rápida (30 minutos)

### Pré-requisitos
- Node.js 16+
- Android Studio
- ANDROID_HOME configurada

### Passos

```bash
# 1. Extraia
unzip ia_hub_mega_final.zip
cd ia_hub_mega

# 2. Instale (se necessário)
npm install

# 3. Compile
npm run build

# 4. Prepare Android
npm run cap:add:android
npm run cap:sync

# 5. Abra no Android Studio
npm run cap:open:android

# 6. No Android Studio: Build > Build APK(s)

# 7. Instale
adb install android/app/build/outputs/apk/debug/app-debug.apk
```
